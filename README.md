## about

Hello.  I work at the intersection of systems programming and security research.  My interests include low‑level software architecture, cryptographic primitives, and the design of resilient digital environments.  I believe that clarity in code leads to clarity in thought.

## projects

**blacknode** — a modular dotfiles framework for arch linux.  each component is isolated, documented, and replaceable.  
[github.com/zhaleff/blacknode](https://github.com/zhaleff/blacknode)

**hollowsec** — a community for security researchers and developers.  we organise ctfs, publish research, and build open source tools.  
[discord.gg/hollowsec](https://discord.gg/hollowsec)
