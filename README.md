## profile

Hello.  I am a security researcher and software architect based on Linux.  My work sits at the intersection of system programming, cryptography, and digital sovereignty.  I believe that understanding how machines operate at the lowest level is essential for building secure, reliable software.

I approach every problem with first principles thinking.  If I do not know something, I learn it.  If a tool does not exist, I build it.  This repository is a reflection of that philosophy – a collection of projects that range from custom desktop environments to security tooling.

## projects

### BlackNode
A modular dotfiles framework for Arch Linux.  Designed for clarity and extensibility – each component is isolated, documented, and replaceable.  It powers my daily driver and serves as a blueprint for others who want to build their own environment.

[![BlackNode](https://img.shields.io/badge/repo-BlackNode-302D41?style=flat-square&logo=github&labelColor=302D41&color=C9CBFF)](https://github.com/zhaleff/BlackNode)

### HollowSec Community
Founder of HollowSec, a collective of security researchers and developers.  We organise CTFs, publish research, and maintain open source tools for the security community.

[![HollowSec](https://img.shields.io/badge/community-HollowSec-5865F2?style=flat-square&logo=discord&labelColor=302D41)](https://discord.gg/hollowsec)

### Cryptographic Tooling
Various libraries and utilities for working with modern cryptographic primitives.  Focus on correct implementations that prioritise security over speed.
